🖼️ PiP Test Page

macOS Spaces Compatible Picture-in-Picture Testing

📖 How to Use

キーボードショートカット:

画像をPicture-in-Pictureで表示: Ctrl+Alt+I

すべてのPiPウィンドウを閉じる: Ctrl+Alt+X

📋 Features

  • macOS Spacesを移動してもPiPウィンドウが表示され続ける
  • NSWindowのcollectionBehaviorを使用した独自PiP実装
  • 画像のPicture-in-Picture表示
  • ドラッグ&ドロップでウィンドウ移動可能
  • ダブルクリックまたは×ボタンで閉じる

🧪 Testing Instructions

Basic Testing:

  1. ページを開いた状態で Ctrl+Alt+I を押す
  2. PiPウィンドウが表示されることを確認
  3. PiPウィンドウをドラッグして移動できることを確認
  4. ×ボタンまたはダブルクリックでウィンドウが閉じることを確認

macOS Spaces Testing:

  1. PiPウィンドウを表示した状態でSpacesを切り替え（Mission Control使用）
  2. 別のワークスペースでもPiPウィンドウが表示されることを確認
  3. 複数のワークスペース間を移動してもPiPウィンドウが常に表示されることを確認

🔧 Technical Details

この実装では以下の技術を使用しています：

  • Qt Framework with macOS-specific Objective-C++ integration
  • NSWindow collectionBehavior for Spaces compatibility
  • Custom PiP window implementation (not HTML5 video PiP API)
  • JavaScript injection for image detection and extraction
  • QWebEngine integration for seamless browser experience
